# Task Tracker CLI

Простой консольный менеджер задач (CLI), написанный на C# в рамках проекта [roadmap.sh](https://roadmap.sh/projects/task-tracker). Приложение позволяет создавать, удалять и отслеживать статус задач, сохраняя все данные в локальный JSON-файл.

## Функционал

- Добавление новых задач.
- Просмотр всех задач или фильтрация по статусу (`todo`, `in-progress`, `done`).
- Обновление описания существующих задач.
- Удаление задач по ID.
- Изменение статуса задачи.

## Требования

- .NET 8.0 SDK или выше.
